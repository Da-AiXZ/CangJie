# 仓颉路线图

## M0 可行性与安装链验证（当前）

- Windows SwiftPM 核心构建与测试
- iPadOS 16.6 SwiftUI 壳
- SQLite/Keychain/流式 HTTP/checkpoint 最小实验
- GitHub Actions iOS 构建与 ad-hoc IPA

## M1 本地优先应用骨架
项目、章节、版本、Provider、任务日志、预算和恢复。

## M2 研究、导入与战略顾问
资料导入/OCR、联网研究、连续访谈、结构化卡片和制作圣经。

## M3 前三章作者校准
审批、拒绝诊断、锁定段落、重写、作者画像和冻结正典。

## M4 滚动自动连载
层级规划、最多五章工作正典、重大事件门、分支与影响分析。

## M5 质量、导出与候选正式版
网文回归、项目包、DOCX、Face ID、性能、安全和正式发布。
